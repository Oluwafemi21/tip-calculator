- I learnt how to use the 'input' event
- How to toggle an active class using Javascript

```html
<div class="grid-tip">

  <button class="btn" value=15>15%</button>

  </div> 
```
```css
.btn.active{
  background-color: var(--Strong-cyan);
  color: black;
  font-weight: 700;
}

```
```js
buttons.forEach(button => {
```
